IVec2

IVec2 extension type

A two-dimensional fixed-point vector.

Prefer pacakge:vector_math over this extension for serious applications; this is a minimalistic alternative that only provides the most basic functionality needed for this library, such as noise generation.

Any tuple of two integers can be cast to a IVec2 using as IVec2:

final vec = (1, 2) as IVec2;

To convert back to a tuple, use the x and y getters, or xy:

final (x, y) = vec.xy;
